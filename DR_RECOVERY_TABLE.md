## 恢复流程

### 1. S3 存储桶恢复

| 步骤 | 操作 | 预计时间 | 负责人 |
|------|------|----------|--------|
| 1 | 识别需要恢复的文件和版本 | 2分钟 | 运维工程师 |
| 2 | 从版本历史中获取版本ID | 1分钟 | 运维工程师 |
| 3 | 执行恢复命令 | 1分钟 | 运维工程师 |
| 4 | 验证数据完整性 | 5分钟 | 运维工程师 |
| 5 | 通知应用团队 | 1分钟 | 运维经理 |
| **总计** | | **10分钟** | |

**恢复命令**:
```bash
aws s3api copy-object \
  --copy-source "miaodong-dev-storage-ee4d262b/file.txt?versionId=VERSION_ID" \
  --bucket miaodong-dev-storage-ee4d262b \
  --key file.txt
```

